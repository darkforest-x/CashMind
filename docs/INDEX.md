## 当前 1.0 结论

1. 先做 Web/PWA，不做原生 iOS App。
2. 无感采集依赖 iPhone 快捷指令，不依赖浏览器后台监听。
3. 服务端部署在 VPS，生产数据落 SQLite。
4. 浏览器/PWA 直接读写个人 VPS 账本；快捷指令密钥用于手机自动写入。
5. 绑定 HTTPS 域名是下一步提升 PWA 体验和安全性的关键动作。
